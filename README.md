# Bitcore-4.0.1

Installation :

```bash
sudo apt install curl 
curl https://raw.githubusercontent.com/creationix/nvm/master/install.sh | bash
source ~/.bashrc
```

Installing Node using NVM

```bash
nvm install 9.11.2
```

Installing Bitcore

```bash
npm -g install bitcore@4.1.0
```

Launch Bitcore

```bash
bitcored
```
